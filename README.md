# CodeGym
A modern neon themed gym landing page. Built with HTML & CSS ONLY. 
This project was created as my first GitHub upload to demonstrate clean structure, responsive design, and front‑end fundamentals.

## Live Demo

## Features
- Fully responsive layout  
- Neon brand theme  
- Hero section with CTA  
- Membership pricing cards  
- Trainer profiles  
- Class branding cards  
- Clean, organised HTML & CSS structure  

## Tech Stack
- HTML5  
- CSS3 (Flexbox + Grid)
